Typography

The typographic base used for Espresso UI is Source Sans Pro set to 1.5rem (15px) with a line height of 1.6 (24px). The Source Sans Pro family contains a large variety of weights to give the designer the most freedom in creating various textures and emphasis for their content.

We have created a consistent family of headings all with distinct letter-spacing line-height and margin. You can create lighter, secondary text in any heading with the .subtext class which is a 1.4rem (14px) with a line-height of 1.6.

Color Palette

The Espresso UI theme has a vibrant and dynamic color palette to give a rich and bold feel to your application's user Interface. The colors can be applied to elements as background colors or accents to text.

A series of descriptive classes make it easy to apply the color for text applications:
.brown .teal .lightgreen .orange .yellow .beige

Applying the teal and lightgreen class to text.

					

					<p> Applying the <span class="teal">teal</span> class to text.</p>

Background colors are just as easy to apply with these classes:
.bg-brown .bg-teal .bg-lightgreen .bg-orange .bg-yellow .bg-beige
